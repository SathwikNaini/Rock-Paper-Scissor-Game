# 🪨 Rock Paper Scissors Game 🎮

A simple Rock Paper Scissors game built with HTML, CSS, and JavaScript. The player competes against the computer in a fun, interactive interface!

## 🚀 Features

- Intuitive user interface with buttons for Rock, Paper, and Scissors
- Random computer choice generation
- Real-time score tracking for user and computer
- Dynamic result display (Win / Lose / Draw)
- Responsive and lightweight design

## 🛠️ Tech Stack

- HTML – Structure of the game
- CSS – Styling and animations
- JavaScript – Game logic and DOM manipulation
